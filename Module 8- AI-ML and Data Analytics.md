**1. Data Collection & Ingestion**

- **Amazon Kinesis**: Real-time streaming data ingestion (Kinesis Data Streams, Kinesis Firehose)

- **Amazon Data Firehose:** option for data ingestion in near real-time. This fully managed service provides automatic provisioning and scaling. It also delivers data within seconds to data lakes, warehouses, and analytics services.

**2. Data Storage**

- **Amazon S3**: Object storage for data lakes

- **Amazon Redshift**: Data warehouse for analytics

**3.Data Catalog**

- **AWS Glue Data Catalog** improves the overall efficiency of data pipelines by delivering metadata to various data stores and analytics services.

**4. Data Processing**

- **AWS Glue**: ETL (Extract, Transform, Load) service with serverless data integration

- **Amazon EMR (Elastic MapReduce)**: Managed Hadoop/Spark clusters for big data processing , Automated

**5. Data Analysis & Query**

- **Amazon Athena**: Serverless SQL queries on S3 data

- **Amazon Redshift**: Complex analytical queries on structured data

- **Amazon OpenSearch Service**: search for relevant content through precise keyword matching or natural language queries.**data visualization** as you analyze and monitor logs, traces, and metrics for various applications.

- **Amazon QuickSight**: Cloud-native BI service for dashboards and visualizations . Amazon Q in QuickSight provides natural language queries so business analysts and users can build, discover, and share meaningful insights in seconds.
